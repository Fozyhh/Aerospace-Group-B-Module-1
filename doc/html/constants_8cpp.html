◆ DT

Real DT
constexpr

Time step size for the simulation.

Controls the temporal resolution of the simulation. Smaller values provide more accurate results but increase computational time. Must satisfy CFL condition for stability. This value is externally defined and can be modified at runtime.

Real RE
constexpr

Reynolds number for the simulation.

Dimensionless number that describes the flow regime (laminar vs turbulent). Represents the ratio of inertial forces to viscous forces within the fluid. This value is externally defined and can be modified at runtime.
