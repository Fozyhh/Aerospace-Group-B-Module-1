Detailed Description

Defines the staggered grid structure for the fluid simulation domain.

This file implements a staggered grid arrangement commonly used in computational fluid dynamics. In a staggered grid, velocity components are stored at cell faces while pressure values are stored at cell centers, improving numerical stability and avoiding checker-board pressure patterns.
