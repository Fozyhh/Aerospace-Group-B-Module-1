\doxysubsection{Detailed Description}
Structure representing the staggered grid arrangement for the fluid domain. 

The grid uses a staggered arrangement where\+:
\begin{DoxyItemize}
\item u-\/velocity components are stored at the x-\/direction cell faces
\item v-\/velocity components are stored at the y-\/direction cell faces
\item w-\/velocity components are stored at the z-\/direction cell faces
\item pressure values are stored at cell vertices
\end{DoxyItemize}
