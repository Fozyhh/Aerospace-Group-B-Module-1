\doxysubsection{Detailed Description}
Core implementation of the incompressible Navier-\/\+Stokes solver. 

This file contains the main solver class for incompressible Navier-\/\+Stokes equations. It handles grid management, parallel computation, boundary conditions, and time-\/stepping solutions using MPI for parallel processing.
