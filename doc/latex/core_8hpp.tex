\doxysubsection{Detailed Description}
Core solver for incompressible Navier-\/\+Stokes equations. 

Contains the main solver class for handling the numerical solution of incompressible Navier-\/\+Stokes equations including the grid, the time step, the final time, the initial condition, the boundary conditions, the source term, the exact solution, the numerical solution, the error, and the output file.
